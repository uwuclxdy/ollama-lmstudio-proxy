# 🦙 → 🖥️ ollama-lmstudio-proxy

Use LM Studio models from anything that speaks the Ollama API. The proxy presents
an Ollama-compatible server and forwards every request to LM Studio.

This wiki is the full technical reference. The
[README](https://github.com/uwuclxdy/ollama-lmstudio-proxy#readme) covers install
and the quickstart; the pages here document every endpoint, flag, and option.
